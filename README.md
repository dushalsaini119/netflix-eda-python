## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset to uncover insights related to content growth, genre distribution, country-wise expansion, and audience targeting. The analysis focuses on transforming raw data into meaningful business insights using Python.

## Dataset
- Source: Kaggle (Netflix Movies and TV Shows dataset)
- Size: ~8,700 records
- Features include content type, release year, date added, country, genre, rating, and duration.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis Performed
- Data cleaning and preprocessing (missing values, date parsing)
- Feature engineering (year_added, month_added, duration_value)
- Content growth analysis over time
- Movies vs TV Shows comparison
- Genre analysis using explode technique
- Country-wise content distribution and localization strategy
- Ratings analysis for audience targeting
- Content lifecycle analysis (release year vs platform addition)
- Genre localization by country
